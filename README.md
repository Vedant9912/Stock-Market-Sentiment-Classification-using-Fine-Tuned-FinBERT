📈 Stock Market Sentiment Classification using Fine-Tuned FinBERT

A financial news sentiment analysis system built using a fine-tuned FinBERT model to classify stock-related news into Positive, Negative, or Neutral sentiments.

This project helps analyze market mood based on financial headlines and can be used for trading insights, research, or decision-support systems.

🚀 Features

Fine-tuned FinBERT model for financial domain

Real-time sentiment prediction

Flask-based web application

Clean and simple frontend (HTML, CSS, JS)

Domain-specific preprocessing

Custom dataset fine-tuning for improved accuracy

🧠 Model Details

Base Model: FinBERT

Task: Multi-class Sentiment Classification

Classes: Positive / Negative / Neutral

Fine-tuned on financial news dataset

Optimized preprocessing for domain-specific vocabulary

🏗️ Tech Stack

Python

PyTorch / Transformers

Flask

HTML, CSS, JavaScript

FinBERT (Pre-trained Financial BERT model)
